## 前端界面选择

### 如何选择v2.0工作流

1. 访问 http://localhost:3000
2. 在"工作流程"区域，点击：
   ```
   🎯 规则拆分 + 人工校准（推荐）
   0.1秒极速拆分 + 可视化校准界面 + 高精度分析
   [98%准确率]
   ```
3. 上传PDF文件
4. 点击"开始拆分"
5. 在校准页面检查并修正题目
6. 点击"确认并继续分析"

### 如何选择v1.0工作流

1. 在"工作流程"区域，点击：
   ```
   ⚡ 直接完整分析（v1.0经典模式）
   LLM自动拆分 + 逐题分析 + 难度素养评估，一键完成
   ```
2. 选择模式（快速/深度）
3. 勾选是否生成PDF报告
4. 点击"开始分析"

## 常见问题

### Q: 为什么看到"开始调用Gemini"？
A: 说明你选择了v1.0工作流，这是正常的。v1.0和v2.0是并存的。

### Q: 如何确认使用了规则拆分？
A: 检查日志中是否有：
- `[INFO] [规则拆分] 开始处理PDF`
- `[INFO] [题号检测] 识别到X个题号`
- `[INFO] [规则拆分] 完成`

### Q: v2.0比v1.0快多少？
A: 拆分阶段快109倍（0.64秒 vs 70秒），总体快1.6倍（13秒 vs 21秒）

### Q: 两种模式可以共存吗？
A: 可以！用户可以根据需求选择不同的工作流。
