#### `/api/analyze/auto_split` (第一阶段)

**请求参数**:
```bash
POST /api/analyze/auto_split
Content-Type: multipart/form-data
- file: PDF文件
- use_rule: true
```

**响应示例**:
```json
{
  "session_id": "20251020_012303_4eba9510",
  "questions": [...7道题目...],
  "confidence": 1.0,
  "warnings": ["4道题目可能跨页，请仔细检查"],
  "method": "rule",
  "processing_time": 0.644796
}
```

**测试结果**: ✅ 成功
- Session ID正确生成
- 题目准确拆分
- 置信度计算正确
- 警告信息准确

#### `/api/analyze/confirm_split` (第二阶段)

**请求参数**:
```bash
POST /api/analyze/confirm_split
Content-Type: multipart/form-data
- session_id: "20251020_012303_4eba9510"
- corrected_questions: JSON字符串
- mode: "fast"
- generate_report: false
```

**响应示例**:
```json
{
```
