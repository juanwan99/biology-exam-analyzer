# 拆题提示词
# Variables: none (image content is provided as separate message parts)
请分析这份初中地理试卷，将其拆分为单独的题目。

初中地理试卷通常包含以下结构：
- **选择题**（第一部分）：通常20-25道，每题2分，共40-50分。每题有A/B/C/D四个选项。常以题组形式出现（2-3道选择题共享一幅地图或材料）。
- **综合题**（第二部分）：通常3-4道，共50-60分。每道大题包含一幅或多幅地图/图表，配合2-5个小问。以读图分析题为主。

返回纯JSON数组格式（不要markdown代码块）：
[
    {
        "id": 1,
        "content": "题目完整文本（包括选项、图表说明等）",
        "image_indices": [0]
    }
]

**拆分规则：**
1. 选择题如果是独立的，每道题单独成题
2. 选择题如果是题组（共享同一幅地图/材料的2-3题），整组归为一个大题
3. 综合题的所有地图/图表描述和小问归入同一道大题
4. 地图、气温降水图、等高线图等图片必须与对应题目一起保留
5. 图片页码（image_indices）准确对应（从0开始）
6. 保留原题号
7. 图表中的图例说明和比例尺信息必须保留
8. 严格返回JSON格式
