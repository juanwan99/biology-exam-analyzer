## 技术改进总结

### 1. Docker镜像增强
```dockerfile
# 新增依赖
fonts-wqy-zenhei          # 中文字体
fonts-wqy-microhei        # 中文字体
locales                   # locale系统
locale-gen zh_CN.UTF-8    # 生成中文locale

# 新增环境变量
ENV LANG=zh_CN.UTF-8
ENV LANGUAGE=zh_CN:zh
ENV LC_ALL=zh_CN.UTF-8
```

### 2. 文件格式智能检测
```python
# 检测文件头部
with open(file_path, 'rb') as f:
    file_header = f.read(4)

if file_header.startswith(b'%PDF'):
    # PDF伪装DOCX，直接使用
    pdf_path = UPLOAD_DIR / f"{file_path.stem}_real.pdf"
    shutil.copy(str(file_path), str(pdf_path))
else:
    # 真实DOCX，转换
    subprocess.run([...], env={'LC_ALL': 'zh_CN.UTF-8'})
```

### 3. LibreOffice转换优化
```python
subprocess.run([
    'libreoffice', '--headless',
    '--convert-to', 'pdf:writer_pdf_Export',  # 显式PDF导出器
    '--outdir', str(UPLOAD_DIR),
    '-env:UserInstallation=file:///tmp/libreoffice_tmp',
    str(file_path)
], check=True, timeout=30, env={'LC_ALL': 'zh_CN.UTF-8'})
```

## v2.0 完整工作流

```
用户访问 http://localhost:3000
    ↓
选择"🎯 规则拆分 + 人工校准（推荐）"
    ↓
上传PDF或DOCX文件
    ↓
[后端] 文件格式检测
    ├─ PDF → 直接使用
    └─ DOCX → LibreOffice转换(中文locale+字体)
    ↓
[规则拆分器] 0.6秒极速拆分
    ├─ 题号识别
    ├─ 边界计算
    ├─ 图片匹配(IoU + proximity)
    └─ 置信度评分
    ↓
返回session_id，跳转到 /correction?session=xxx
    ↓
[人工校准页面] GET /api/analyze/session/{session_id}
    ├─ 显示拆分结果
    ├─ 置信度标识(红/黄/绿)
    ├─ 警告信息
    └─ 编辑/合并/拆分/删除操作
    ↓
用户确认 → POST /api/analyze/confirm_split
    ↓
[完整分析]
    ├─ 逐题分析
    ├─ 难度评估
    ├─ 素养分析
    └─ 生成PDF报告(可选)
    ↓
显示结果在当前页面
```
